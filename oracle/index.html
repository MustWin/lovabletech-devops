Case Study: Oracle Terraform Provider

Learn how we utilize microservices and DevOps best practices to support operations at Oracle

Summary of Oracle Terraform

Oracle released a new public cloud offering called Baremetal. In order to increase adoption, they wanted to enable access to the platform using the popular Hashicorp tool Terraform. Lovable Tech built a Golang SDK and the actual terraform provider code that is now public on Github, we also provided 6 months of community support to handle bug reports and encourage engagement
